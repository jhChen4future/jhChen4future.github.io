# Empirical and Causal Operations

## Start with the estimand
Write down:
- unit;
- treatment or exposure;
- outcome;
- comparison/counterfactual;
- population;
- time horizon;
- estimand;
- assumptions required.

## High-value branches
- Identification audit
- Data-generating-process simulation
- Alternative variable construction
- Negative controls or placebo tests
- Specification curve or multiverse
- Sensitivity to omitted variables
- Attrition/missingness analysis
- Heterogeneity with pre-specified moderators
- Predictive versus causal interpretation audit

## Anti-degradation rule
Do not respond to a weak baseline result by mechanically adding controls, fixed effects, interactions, or alternative models. Diagnose which assumption each addition addresses and whether it changes the estimand.

## Progress artifact
Leave an estimand statement, assumption table, diagnostic script, robustness matrix, or decision about the credible claim.
