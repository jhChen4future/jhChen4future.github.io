# Literature and Theory Operations

Use literature work to alter the research state, not merely enlarge a bibliography.

## Branches
- Concept genealogy: origins, changes in meaning, and later reinterpretations.
- Rival vocabulary: different terms for the same phenomenon.
- Rival framework: incompatible mechanisms or taxonomies.
- Boundary evidence: conditions under which a result disappears or reverses.
- Measurement critique: whether canonical tasks identify the construct.
- Method migration: how adjacent fields operationalize the problem.
- Frontier: recent working papers, replications, meta-analyses, and debates.

## Source hierarchy
Prefer original articles, official working-paper repositories, datasets, code repositories, and authoritative documentation. Use social media and commentary for discovery, author clarification, and debate mapping—not as sufficient evidence for scholarly claims.

## Information-gain test
A source is load-bearing only if it:
- changes a claim or its confidence;
- adds a distinct mechanism;
- challenges a boundary;
- resolves a disagreement;
- improves measurement or identification;
- traces an idea's origin;
- changes the research agenda.

## Output
Organize by problem or theoretical conflict, not as an annotated list. Explicitly mark what is new relative to the inherited project state.
